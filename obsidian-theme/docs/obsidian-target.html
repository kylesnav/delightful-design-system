Features

The design system uses a neo-brutalist approach with warm OKLCH colors:

  • Solid shadows with zero blur
  • Structural 2px borders throughout
  • Playful border radii for personality
  • Six accent color families

Typography

Headings use tight tracking with heavy weight. Body text uses comfortable leading for readability.
